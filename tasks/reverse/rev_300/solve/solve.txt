ELF 64bit static linked C++ based.

Основная задача найти табличку виртуальных функций среди которых 3 функции проверяющий флаг по частям

Первая часть - проверка на размер и формат флага.
Вторая часть - проверка первых 32 символов внутри флага путём XOR'а с рандомно генерируемой строкой, которая генерируется с помощью LCG для решения которого всё дано.
Третья часть - полученная последовательность из LCG складывается и берётся по модулю 256 в итоге получается один ключ для XOR'a

При первой загрузке в IDA задание выглядит очень страшно, но более-менее средненький ревёрсер спокойно разберёт за 1 часа в отладчике.
